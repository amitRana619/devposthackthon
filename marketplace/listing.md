# Marketplace Listing Draft

## Product Name

Energy Ops Agent

## Short Description

AI energy-operations agent for B2B facilities teams handling extreme weather and
utility peak-demand pricing conflicts.

## Long Description

Energy Ops Agent helps B2B building operators protect occupants and critical
zones while still responding to utility demand-response events. The agent
combines grounded weather, pricing, occupancy, and building-load context to
produce source-backed energy plans with observability traces.

The product is built with Google ADK and exposes a Cloud Run-ready API for
enterprise deployment. Tool access is mediated through MCP and deterministic
service logic enforces safety-first conflict resolution, source preservation,
load-level demand-response allocation, and evaluation gates.

## Buyer Value

- Reduce unsafe or delayed decisions during rare grid and weather events.
- Preserve occupant safety and critical-zone comfort before cost optimization.
- Estimate demand-response cost avoidance and protected business risk.
- Return named controllable loads selected for shedding.
- Preserve source IDs for auditability.
- Swap sample adapters for BMS, utility tariff, weather, and occupancy systems.

## Deployment Model

SaaS product hosted primarily on Google Cloud, with optional customer-project
data plane adapters for private business systems.

Support channels, response targets, escalation process, and customer success
